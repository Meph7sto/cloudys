# Cloudys 主要接口文档

> 2026-06-24 | 聚焦核心业务接口

- **网关入口**: `http://localhost:8008`
- **认证**: Header `Authorization: Bearer <token>`
- **响应格式**: 统一信封 `{ code, message, data }`

---

## 服务与接口总览

### gateway-service（网关，8008）
统一鉴权、限流、路由分发。所有接口均经由网关访问。

### eureka-service（注册中心，8761）
Netflix Eureka，服务注册与发现，无业务接口。

## 服务间调用拓扑

```
auth-service        ← OpenFeign ← requirement-service
                    ← OpenFeign ← requirement-analysis-service

project-service     ← OpenFeign ← requirement-service

requirement-service ← OpenFeign ← requirement-analysis-service

inference-service   ← OpenFeign ← requirement-analysis-service
    ↓ (WebClient)
python-sidecar (:8000)
    ↓ (HTTP)
DeepSeek API（外部 LLM）
```

---

## 核心业务流程

```
创建项目 → 录入转写(SSE) → AI 提取 L1-L3 → AI 生成 L4
    → 导入需求 → 分类 & 去重 → 冲突检测 → 可追溯性分析
    → 创建基线 → 评审 → 变更控制
```
